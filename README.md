# memory-game
Project Description:
A short description of your game.
Match pictures and find out how good is your memory.

user stories:
User can see a grid with n x n cards. All the cards are faced down initially.
User can click a button to start the game. When this button is clicked, a timer will start.
User can click on any card to unveil the image that is underneath it . The image will be displayed until the user clicks on a 2nd card. If there is a match, the 2 cards will be eliminated from the game. If there isnâ€™t a match, the 2 cards will flip back to their original state.
